Interactions APIs

Events

Core click-streams

Name Description
product_detail_page_view Used when a user views the detail page of a product.
category_page_view Used when a user views a category page for a specific "family" or "group" or products or content.
search Used to record a search event with the keywords and filters the user used.
impression Used to record when a user saw or was presented with a product or content asset.

Conversion (eCommerce)

Name Description
add_to_cart Used when a user adds a product into their shopping cart.
remove_from_cart Used when a user removes a product from their shopping cart.
subscribe Used when a user subscribes a product for its information.
checkout Used when a user enters checks out with a set of products.
refund Used when a user requests a refund of products they bought.

Consumption (content media)

Name Description
read Used to record when and for how long a user reads a piece of written content.
watch Used to record when and for how long a user watches content that is of a video format.
listen Used to record when and for how long a user listens to content that is of an audio format.
add_to_collection Used when a user adds a product to their personal collection.
remove_from_collection Used when a user removes a product from their personal collection.

Feedback signals

Name Description
like Used to record when a user indicates a like for a product.
dislike Used when a user indicates a dislike for a product or indicates they would like to not be recommended content or products like this in the future.
share Used when a user shares a product or piece of content.
rate Used when a user gives a rating to a product or piece of content.
bookmark Used when a user bookmarks a product.

Additional click-streams

Name Description
home_page_view Used when a user views your home page.
promo_page_view Used when a user views a specific promotional or curated marketing page about certain products or content.
product_image_view Used when a user views the image of a product (e.g. to enlarge a product photo).

Custom

Name Description
custom Used when you want to record any custom interaction events between users and products.
