    Guide for building your ask module in answers page

    There are three ways to build your ask module:

    The standard UI solution

    • Features are identical to the demo.
    • Can be done with 2 lines of HTML.
    • Limited customization: CSS, texts.
    • Follow the guide.

    The custom UI solution

    • Fully customizable: HTML, CSS, texts, logic.
    • Follow the guide.

    Build from scratch

    You can also build your UI from scratch, without the assistance of SDK UI elements. Read the API section for details.
