Install

Include Miso Client SDK on your website.

<script src="https://.../client-sdk@latest"></script>
const miso = window.miso('your-publishable-api-key');

If you had the SDK script loaded asynchronously, you can access miso in the following callback.

<script async src="https://.../client-sdk@latest"></script>
const misocmd = window.misocmd || (window.misocmd = []); misocmd.push(function() { const miso = new MisoClient('...'); });

Config

Specify user_id and user_hash once, and they will be passed into API payloads automatically.

  • You can clear or update the values later.
  • You can override the values per API call.
  • Miso SDK takes care anonymous id automatically.
miso.context.user_id = 'user_id'; miso.context.user_hash = 'user_hash';

Search

Search for products using search/search API.

const response = await miso.api.search.search({ q: 'doge' }); const products = response.products;

Recommendation

Get personalized recommendation from recommendation/user_to_products API.

const response = await miso.api.recommendation.userToProducts({ }); const products = response.products;
