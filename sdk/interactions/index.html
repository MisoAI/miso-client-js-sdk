Interactions APIs

Events

Clickstream

Name Description
Used when a user views the detail page of a product.
Used when a user views a category page for a specific "family" or "group" or products or content.
Used to record a search event with the keywords and filters the user used.
Used to record when a user saw or was presented with a product or content asset.
Used when a user views your home page.
Used when a user views a specific promotional or curated marketing page about certain products or content.
Used when a user views the image of a product (e.g. to enlarge a product photo).

Conversion (eCommerce)

Name Description
Used when a user adds a product into their shopping cart.
Used when a user removes a product from their shopping cart.
Used when a user subscribes a product for its information.
Used when a user enters checks out with a set of products.
Used when a user requests a refund of products they bought.

Consumption (content media)

Name Description
Used to record when and for how long a user reads a piece of written content.
Used to record when and for how long a user watches content that is of a video format.
Used to record when and for how long a user listens to content that is of an audio format.
Used when a user "complete" a product (e.g. complete a course or a video).
Used when a user adds a product to their personal collection.
Used when a user removes a product from their personal collection.

Feedback signals

Name Description
Used to record when a user indicates a like for a product.
Used when a user indicates a dislike for a product or indicates they would like to not be recommended content or products like this in the future.
Used when a user shares a product or piece of content.
Used when a user gives a rating to a product or piece of content.
Used when a user bookmarks a product.

Custom

Name Description
Used when you want to record any custom interaction events between users and products.
