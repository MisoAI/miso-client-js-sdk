Install

Install Miso client JS SDK on your website.

<script src="https://some.cdn.io/miso.js"></script>

If you had the SDK script loaded asynchronously, use miso functions in the callback.

<script async src="https://some.cdn.io/miso.js"></script>
var miso = window.miso || (window.miso = []); miso.push(function() { // miso functions are available here });

Config

Set API key for any future API calls.

miso.config({ api_key: '...' });

Configured user_id, anonymous_id, and user_hash will be passed into API payloads as well.

  • However they can be overriden per API call.
miso.config({ anonymous_id: 'miso-client-sdk-demo-anonymous', user_id: 'miso-client-sdk-demo-user', user_hash: '...' });
